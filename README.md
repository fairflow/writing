## Local Development

To run this site locally:

```bash
bundle install
bundle exec jekyll serve
```

Then visit `http://localhost:4000/writing/`

## Structure

- `_config.yml` - Jekyll configuration
- `index.md` - Homepage
- `_tools/` - Individual tool documentation pages
- `tools/index.md` - Tools directory listing
